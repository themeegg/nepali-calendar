== Description ==

This is a simple Nepali calender plugin plus calender widget.

Shortcode of this plugin

*   [nepali-calendar]


Please visit our official page <a href="http://docs.themeegg.com/docs/nepali-calendar/" target="_blank">http://docs.themeegg.com/docs/nepali-calendar/</a> for more information.

Get free support at http://support.themeegg.com

= Requirements =

* WordPress 4.5 or later.

= Contribute =

You can contribute to the source code in our [GitHub](https://github.com/themeegg/nepali-calendar/) page.

== Installation ==

1. Download the plugin and extract the files
2. Upload "nepali-calendar" to your wp-
content/plugins/directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Put shortcode to anywhere.
